Quick Start

Get started in minutes

1. Install PortPilot

npm install -D portpilot

2. Create .pilotrc.js

export default {
  port: 3000,
  domain: 'myapp.local',
  isHttps: false,
  isLog: true
}

3. Run PortPilot

npx portpilot
